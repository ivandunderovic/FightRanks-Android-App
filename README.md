# FightRanks

FightRanks is an Android application built with Kotlin and Jetpack Compose for browsing MMA/UFC fighter rankings by their weight class.
This project was made as a part of a university coursework project.

## Features

- Browse fighters by category
- View fighter details
- Add and remove favorites
- Search favorites
- Firebase Firestore integration
- Firebase Authentication login

## Technologies used

- Kotlin
- Jetpack Compose
- Firebase Firestore
- Firebase Authentication
- Navigation Compose

## Screens

- Home screen with weight classes
- Fighters screen with ranking list
- Favorites screen with search
- Sign In screen

## Project Structure

- `ui/` – Compose screens and UI components
- `data/` – models and ViewModel for fighters
- `viewmodel/` – authentication logic
- `navigation/` – app routes and navigation setup
